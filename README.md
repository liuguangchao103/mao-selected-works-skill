## 概述

本 Skill 基于教员选集四卷（1925-1949年）157篇重要文献，覆盖中国革命的关键理论与实践。

## 核心内容

### 哲学思想
- **矛盾论** - 对立统一法则，矛盾普遍性与特殊性
- **实践论** - 认识与实践的关系
- **反对本本主义** - 调查研究方法

### 政治策略
- **阶级分析** - 分清敌友是革命首要问题
- **统一战线** - 团结一切可以团结的力量
- **新民主主义论** - 中国革命两步走战略
- **论持久战** - 抗日战争三阶段论

### 军事思想
- **人民战争** - 兵民是胜利之本
- **十大军事原则** - 集中优势兵力，各个歼灭敌人
- **游击战术** - 敌进我退，敌驻我扰，敌疲我打，敌退我追

### 领导方法
- **群众路线** - 从群众中来，到群众中去
- **实事求是** - 一切从实际出发
- **批评与自我批评** - 知无不言，言无不尽

### 治国理政
- **两个务必** - 谦虚谨慎、艰苦奋斗
- **为人民服务** - 全心全意为人民服务
- **论人民民主专政** - 人民民主专政理论
- **七届二中全会** - 工作重心由乡村转向城市

- 毛选、教员思想

- 阶级分析、统一战线、新民主主义
- 论持久战、群众路线
- 七届二中全会、两个务必
- 为人民服务、论人民民主专政
- 湖南农民运动
- 十大军事原则
