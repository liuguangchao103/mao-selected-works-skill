# 哲学思想

## 矛盾论（1937年8月）

### 核心观点

**对立统一法则**
事物发展的根本原因在于内部矛盾性。对立统一法则是唯物辩证法的最根本法则。

**两种宇宙观**
- 形而上学：孤立、静止、片面观点看待世界
- 辩证法：对立统一观点看待世界

**矛盾普遍性**
- 矛盾存在于一切事物发展过程中
- 每一事物发展过程存在自始至终的矛盾运动

**矛盾特殊性**
- 不同质的矛盾用不同质的方法解决
- 主要矛盾起领导作用，次要矛盾处于次要和服从地位
- 矛盾主要方面决定事物性质

**同一性与斗争性**
- 同一性：矛盾双方互相依存，共处于一个统一体中；互相转化
- 斗争性：贯穿过程始终，是绝对的
- 有条件的相对的同一性和无条件的绝对的斗争性相结合，构成一切事物的矛盾运动

**对抗在矛盾中的地位**
- 对抗是矛盾斗争的一种形式，而非一切形式
- 阶级社会中革命战争不可避免

### 认识论要点

"矛盾存在于一切客观事物和主观思维的过程中，矛盾贯串于一切过程的始终"

"矛盾的共性个性、绝对相对的道理，是关于事物矛盾的问题的精髓，不懂得它，就等于抛弃了辩证法"

## 反对本本主义（1930年5月）

### 核心观点

**没有调查，没有发言权**
"你对于某个问题没有调查，就停止你对于某个问题的发言权"

**调查就是解决问题**
"调查就像'十月怀胎'，解决问题就像'一朝分娩'"

**本本主义必须纠正**
"马克思主义的'本本'是要学习的，但是必须同我国的实际情况相结合"

**中国革命胜利要靠中国同志了解中国情况**
"中国革命斗争的胜利要靠中国同志了解中国情况"

### 调查研究方法

1. 开调查会作讨论式的调查
2. 调查会要选对的人：老年人（有经验）、青年人（有进步思想）
3. 定调查纲目：有大纲和细目
4. 亲身出马，不能单靠书面报告
5. 要深入
6. 自己做记录
